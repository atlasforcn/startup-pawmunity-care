# Pawmunity Care Platform

這個 repo 是依據 `Pawmunity寵物共養社群系統` 在教育部大專校院創新創業實戰模擬學習平臺中的成功案例概念做出的前端原型。原型把「都市寵物臨時照護、可信任共養者、Hub 安心中心、PawScore 信任評分」實作成一個寵物共養照護媒合平台。

## 比賽來源

- 比賽/計畫：大專校院創新創業實戰模擬學習平臺
- 年度：114 年
- 屆次/階段：第二梯次成功案例
- 學校：長庚大學
- 團隊名稱：Pawmunity
- 得獎/成功案例作品：Pawmunity寵物共養社群系統
- 主辦：教育部
- 官方來源：https://ssp.moe.gov.tw/cases/1518

## 核心概念

都市寵物飼主常在工作、出差或突發狀況下需要臨時照護者，但傳統寄宿和親友協助常有彈性不足、信任不透明、照護風險集中等問題。這個原型把概念拆成五個核心流程：

- 照護需求刊登：飼主建立寵物資料、照護時段、特殊需求與地點。
- 共養者媒合：依照距離、可服務時段、照護經驗與 PawScore 推薦照護者。
- PawScore 信任評分：累積照護紀錄、回饋、驗證項目與風險提示。
- Hub 安心中心：在緊急狀況或交接時提供實體支援點。
- 照護日誌：照護者回報餵食、散步、用藥、照片與異常事件。

## 原型範圍

這是靜態前端原型，資料皆為模擬。它不代表原團隊正式產品，也未使用原團隊未公開資料、商標素材或後端服務。

## 使用方式

用瀏覽器開啟 `index.html`，或透過主整理網站的原型連結進入。
